[Docusaurus](https://docusaurus.io) là một trình tạo trang web tĩnh mã nguồn mở giúp chuyển đổi các file Markdown thành một trang web tài liệu. Được tạo bởi Facebook, Docusaurus được viết bằng ngôn ngữ lập trình Node.js. Hàng ngàn tổ chức sử dụng Docusaurus để vận hành các trang web tài liệu của họ. Một ví dụ về trang web được tạo bởi Docusaurus được hiển thị bên dưới.

Docusaurus sử dụng bộ xử lý Markdown [remarkable](https://github.com/jonschlinkert/remarkable).
